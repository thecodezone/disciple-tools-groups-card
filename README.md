# Disciple.Tools - Groups Tile

This is a simple plugin that adds a group tile to the Disciple.Tools dashboard. It provides a simple way to search groups, filter groups by user, and view a groupâ€™s info and health.

## Requirements

- Disciple.Tools Theme installed on a Wordpress Server

## Installing

- Install as a standard Disciple.Tools/Wordpress plugin in the system Admin/Plugins area.
- Requires the user role of Administrator.
